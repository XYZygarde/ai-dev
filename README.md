# ai-dev

Development workspace for **Flyrank AI** — FlyRank AI is a Chicago-based technology and growth platform that automates organic search and AI-engine visibility for direct-to-consumer (D2C) and SaaS brands. It helps companies optimize their online content so that search algorithms and AI models (like ChatGPT, Perplexity, and Gemini) can easily discover, trust, and cite their products.

This repository is in early setup. Application code, APIs, and deployment docs will be added here as the project grows.
